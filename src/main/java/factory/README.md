# 工厂设计模式

    工厂模式是我们最常用的实例化对象模式了，是用工厂方法代替new操作的一种模式。
    
    实现了创建者与调用者的分离；
    
    详细分类：
        简单工厂模式；
        工厂方法模式；
        抽象工厂模式；
